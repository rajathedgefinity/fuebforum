community.views module¶

community.views.closed_issues(request)¶

This Function declaration has made for Listing the total closed issues that have been recorded from the Users.

community.views.counting_opened_issues(request)¶

This Function declaration has made for counting the total Opened and Closed issues that have been recorded from the Users.

community.views.faq(request)¶

This Function declaration is created for loading the FAQ[Frequently Asked Questions] Page.

community.views.forum_thread(request, id)¶

This is the function which used to open the topic particularly the user clicked on the landing page

community.views.index(request)¶

This function loads the Landing page Where here we can navigate to each topics that has been created for the discussion.

At the same time once you got redirected from the signin page it’ll be redirected here from the Signin page with Active User.

community.views.login_forum(request)¶

This Function declaration is for created for loading Login Page and for Accepting the Credentials, for Login.

community.views.login_register(request)¶

This Function declaration is for created for loading SignUp Page.

community.views.logout_user(request)¶

This Function declaration is for created for logout if their is a Active User and if logged In. To Checkout i’ve been used decorator @login_required

community.views.opened_issues(request)¶

This Function declaration has made for Listing the total Opened issues that have been recorded from the Users.

community.views.profile(request)¶

This function is defined to load the topics created by that particular User. Only If and if user is logged In.

community.views.profile_settings(request)¶

This function is defined to load the Profile Settings Page. Only If and if user is logged In.
